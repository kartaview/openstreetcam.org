Requirements:

Needs NodeJS+NPM and node TypeScript 2.

Install:
1)npm install -g @angular/cli
2)npm install


Run:
ng serve

Build
ng build // debug, uses environments/environment.ts
ng build -prod // no debug, uses environments/environment.prod.ts
ng build --env=testing

Test:
http://localhost:4200/


Environment settings:
src/environments/
